<span data-ttu-id="8f92d-201">Das Verzeichnis *Logs* (Protokolle) kann mit einem der folgenden beiden Ansätze für die Bereitstellung erstellt werden:</span><span class="sxs-lookup"><span data-stu-id="8f92d-201">A *Logs* directory can be created for the deployment using one of the following two approaches:</span></span>
