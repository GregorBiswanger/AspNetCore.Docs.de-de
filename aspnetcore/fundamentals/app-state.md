## <a name="session-state"></a><span data-ttu-id="4fc78-145">Sitzungszustand</span><span class="sxs-lookup"><span data-stu-id="4fc78-145">Session state</span></span>

<span data-ttu-id="4fc78-153">Der Sitzungszustand verhält sich wie folgt:</span><span class="sxs-lookup"><span data-stu-id="4fc78-153">Session state exhibits the following behaviors:</span></span>
